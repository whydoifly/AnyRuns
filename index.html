Frequently Asked Questions

Have any questions about the AnyRuns app?

See below to find out if we have already answered it.

How can I install the app ?
You can install AnyRuns via Google Play or the Apple Store. All you need to do is type “AnyRuns” into the search bar, find the app and install it (for free!).
Where are the settings options?
Launch the app and press on your Profile (bottom right-hand side). Once this has loaded, you will find the Settings option, which will allow you to make changes to your profile.
How can I manage the settings?
See above. Once you have found the Settings option, you will be able to change details including your profile name, icon photo and address. You can also reset your password here.
Is there any authentication system?
We will ask you to make an AnyRuns account using a valid email address and password. Your account will be protected, with any suspicious activity resulting in it being locked – if this happens, we will send you a password reset email.
How can I start receiving notifications?
To receive AnyRuns notifications on your device, you must authorise it via the app’s Settings. All you need to do is press ‘Approve Notifications’ to receive updates about your delivery/collection, as well as any special offers.
